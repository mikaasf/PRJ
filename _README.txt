Sistema Anotação de Vídeo com dados fisiológicos para análise de emoções

Bibliotecas necessárias para o funcionamento do projeto:
 - pyrealsense2
 - numpy
 - cv2
 - scipy
 - pyaudio
 - deepface
 - flask
 - flask_socketio

No ato de instalação da biblioteca deepface, será necessário instalar um modelo já treinado, indicado num link.
